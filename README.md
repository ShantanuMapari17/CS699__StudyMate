## Instructions for postgres
### Installation Link
`https://www.postgresql.org/download/linux/ubuntu/`

### Some Commands to start postgres
- changing the user
``` $ sudo -i -u postgres```

- running psql
  ``` $ psql```
- exit from psql shell
  `>> \q`

-Create dB
`createdb my_pgdb`

-Switching to database
`psql -d my_pgdb`

-Getting current info 
`\conninfo`
